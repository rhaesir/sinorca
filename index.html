Introdução

Esta é a versão dois de Sinorca, uma revisão do design original que enviei ao OSWD em janeiro de 2003. A seguir está um resumo das diferenças principais entre esta versão e a original.

Sinorca é baseado no site da Acronis, uma empresa norte-americana de software. Este template está marcado em XHTML 1.0 Strict e estilizado com CSS, permitindo maior compatibilidade e flexibilidade.

Você pode usar este design para qualquer finalidade — pessoal, comercial ou outra — sem obrigação. Se publicar um site baseado neste template, por favor me avise.

Compatibilidade entre navegadores

Esta versão foi testada nos navegadores Opera, Gecko (Mozilla) e Internet Explorer para garantir renderização consistente.

^ TOPO

Folhas de estilo modificadas

Folha de estilo de tela mais completa
Parâmetros de estilo são especificados explicitamente para aumentar consistência.
Remoção da folha de esquema de cores separada
As definições de cor foram incorporadas à folha de estilo principal.
Folha de estilo para impressão
Agora há uma folha de estilo própria para impressão em preto e branco.

Acessibilidade aprimorada

Esta versão possui melhorias de acessibilidade e pode ser transformada para agentes não visuais como leitores de tela e braille. O layout é adaptável ao tamanho da fonte definido pelo usuário.
